## Descripción del Proyecto

**Sitio Web Pintura** es un proyecto que consta de un **Sitio Web** que desarrollé para una nueva gama de pinturas presentada por una pinturería específica. Este sitio web fue desarrollado con **WordPress** y **Elementor**.

Es un sitio web sencillo que muestra los nuevos productos y sus beneficios. Además, incluye funcionalidades básicas como un formulario de contacto. Lo que destaca de este proyecto es la efectiva utilización de Elementor, el diseño atractivo y el uso de animaciones en la página principal.

Puedes acceder al sitio web aquí: [Toulouse Pinturas](http://toulousepinturas.com.ar/).
