## Descripción del Proyecto

**Sorteos Webapp** es una aplicación web diseñada para que una empresa realice sorteos en línea, permitiendo a los usuarios cargar sus datos personales y los detalles de un producto específico para participar.

### Funcionalidad

1. **Registro del Cliente:**
   - El usuario ingresa su email.
   - Si el email no existe en la base de datos, se registra al usuario con sus datos personales.

2. **Participación en el Sorteo:**
   - El usuario introduce el código y la hora del producto.
   - El usuario participa instantáneamente en un sorteo donde puede ganar uno de los premios preconfigurados.

3. **Sorteo Global:**
   - Todos los usuarios participan en un sorteo global, el cual se realiza cuando se llega a la fecha de finalización del sorteo.

### Desafíos

El desarrollo de **Sorteos Webapp** presentó varios desafíos significativos:

- **Plazos Ajustados:** El proyecto debía completarse en un tiempo limitado, siguiendo el diseño proporcionado por el equipo de UX.
- **Generación de Informes:** Fue necesario desarrollar funcionalidades para generar informes en Excel, permitiendo a la empresa recopilar y analizar los datos de los participantes de manera eficiente.

### Conclusión

**Sorteos Webapp** es una solución eficaz para la realización de sorteos en línea, combinando una experiencia de usuario intuitiva con la capacidad de manejar grandes volúmenes de datos. Este proyecto demuestra la habilidad de integrar múltiples funcionalidades en una plataforma cohesiva, cumpliendo con plazos estrictos y requisitos específicos de diseño.
