## Descripción del Proyecto

El **Sistema de Estacionamientos** es una aplicación web diseñada para gestionar eficientemente un negocio de estacionamientos. Este sistema permite administrar las entradas y salidas de vehículos, gestionar estadías mensuales y semanales, registrar movimientos de dinero personalizados, y generar informes detallados. Además, incluye funcionalidades de facturación a través de impresora fiscal y facturación electrónica.

El sistema está concebido para ser multiusuario, facilitando la gestión tanto para operadores, que se encargan de la entrada y salida de vehículos, como para administradores, que manejan informes y estadías prolongadas.

## Funcionalidad

### Operadores
Los operadores pueden:

- **Registrar entradas y salidas de vehículos** de manera rápida y eficiente.
- **Emitir tickets personalizados** mediante impresoras fiscales.
- **Gestionar estadías temporales** de vehículos.

### Administradores
Los administradores tienen acceso a:

- **Gestión de usuarios**: Crear, editar y eliminar cuentas de operadores.
- **Informes y estadísticas**: Generar informes detallados sobre las operaciones diarias, mensuales y anuales.
- **Facturación**: Emitir facturas electrónicas a través de la API de AFIP y gestionar la facturación a través de impresoras fiscales.
- **Configuración de tarifas**: Establecer y ajustar tarifas para diferentes tipos de estadías.

## Desafíos

### Integración con Impresoras Fiscales
Uno de los principales desafíos fue integrar el sistema con impresoras fiscales para la emisión de tickets personalizados, lo que requirió un desarrollo técnico específico para asegurar la compatibilidad y funcionalidad eficiente.

### Implementación de la API de AFIP
La implementación de la API de AFIP para la facturación electrónica presentó desafíos técnicos significativos, asegurando el cumplimiento de las normativas locales y la integración fluida con el sistema de facturación del negocio.

### Diseño de una UI Ágil
Desarrollar una interfaz de usuario (UI) que permitiera un uso rápido y eficiente del sistema fue crucial, especialmente para manejar las horas pico en las que los vehículos entran y salen con gran frecuencia. Esto implicó un enfoque en la usabilidad y en la minimización del tiempo requerido para registrar cada operación.

## Conclusión

El **Sistema de Estacionamientos** se ha demostrado ser una solución robusta y versátil para la gestión de estacionamientos, facilitando tanto las operaciones diarias como la administración a largo plazo. La combinación de funciones avanzadas y una interfaz de usuario intuitiva permite optimizar el flujo de trabajo y mejorar la experiencia de los usuarios, tanto operadores como administradores.
