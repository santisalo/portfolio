## Descripción del Proyecto

**Tienda Wpp** es una aplicación web compuesta por un **Panel de Control privado** y algunas rutas públicas para los usuarios finales. Está desarrollada con **Vue.js** y **Laravel**. Para la parte visual se utilizó el framework **Vuetify**.

Esta aplicación está diseñada para facilitar su uso sin necesidad de descargar una aplicación específica. Los usuarios finales pueden acceder al catálogo simplemente escaneando un código QR. Además, está pensada tanto para **uso en el local físico** como para **realizar pedidos online por WhatsApp**. El panel de control permite configurar todas estas funcionalidades.

## Funciones

Los usuarios que gestionan el negocio pueden acceder al panel de control con su usuario y contraseña para gestionar su catálogo de productos.

**Dentro del panel de control, las funcionalidades disponibles incluyen:**

- **Creación, edición y eliminación de productos**
- **Creación, edición y eliminación de categorías**
- **Configuración de datos del negocio**, incluyendo redes sociales y número de WhatsApp para pedidos
- **Configuración de gama de colores** para mantener el estilo del negocio

**En la parte pública de la aplicación, los usuarios finales pueden:**

- **Visualizar los datos del negocio** en la cabecera
- **Visualizar los productos del negocio** agrupados por categoría
- **Realizar búsquedas de productos**
- **Armar un pedido** con diversos productos, cantidades, variantes y extras
- **Generar un mensaje de WhatsApp** con el pedido y los datos del cliente

## Desafíos

Durante el desarrollo del proyecto, se presentaron diversos desafíos a nivel de arquitectura, diseño e implementación, tales como:

- Gestionar múltiples negocios bajo una misma aplicación web
- Personalizar cada una de las tiendas según las necesidades de cada cliente/negocio
- Crear una interfaz amigable para el usuario final
- Desarrollar una interfaz autogestionable y amigable para el usuario que cargará los productos
- Implementar la carga masiva de productos mediante archivos Excel
- Generar etiquetas QR estilizadas en formato JPG, disponibles para descargar, con la marca de cada negocio

---

**Tienda Wpp** es una solución integral y versátil para la gestión de negocios, que combina facilidad de uso y personalización. La capacidad de acceder a catálogos y realizar pedidos de manera eficiente, tanto en el local físico como online, resalta la efectividad del sistema.
