## Descripción del Proyecto

**App Validación por GPS** es un proyecto que consta de un **Panel de Control** para administradores y una **Aplicación Móvil** para usuarios finales. Es utilizada por los administradores del sistema para que los usuarios validen su geolocalización al realizar trabajos en lugares específicos a horas determinadas.

## Funcionalidad

### Panel de Control para Administradores
A través del panel de control, los administradores del sistema pueden:

- **Crear, editar y eliminar usuarios**: Gestionar cuentas de usuario.
- **Visualizar todas las órdenes de trabajo en un mapa**: Monitorear las tareas asignadas.

### Aplicación Móvil para Usuarios
Mediante la aplicación móvil, los usuarios pueden:

- **Iniciar sesión con sus datos**: Acceder a su cuenta personal.
- **Cargar una nueva orden de trabajo**: Registrar nuevas tareas.
- **Validar su geolocalización con huella dactilar**: Confirmar su ubicación mediante huella dactilar, incluso sin conexión a internet.
- **Envío de datos asíncrono**: Enviar datos de validación una vez que se restablezca la conexión a internet.

## Desafíos

El desarrollo de **App Validación por GPS** enfrentó varios desafíos significativos:

- **Funcionalidad offline**: Garantizar que el sistema pudiera almacenar datos localmente en el teléfono y sincronizarse con la base de datos una vez detectada una conexión a internet.
- **Seguridad biométrica**: Implementar un sistema de validación biométrica confiable para garantizar la identidad del usuario al validar tareas. La aplicación debía detectar cuándo los usuarios agregaban nuevas huellas dactilares a sus dispositivos y solicitarles que se registraran nuevamente en la aplicación para evitar validaciones no autorizadas.

## Conclusión

**App Validación por GPS** es una solución robusta y versátil para la validación de geolocalización en la gestión de tareas. La aplicación mejora la eficiencia y seguridad al permitir que los usuarios validen su presencia en lugares específicos y horas determinadas utilizando datos biométricos, incluso sin conexión a internet. Los administradores se benefician de un panel de control completo para gestionar usuarios y monitorear órdenes de trabajo de manera efectiva. Este proyecto demuestra la capacidad de integrar tecnologías avanzadas para crear una plataforma confiable y segura que satisfaga las necesidades tanto de los administradores como de los usuarios finales.
