## Descripción del Proyecto

**Sistema Hotelería** es un sistema web diseñado como una solución integral para la gestión hotelera. Ofrece una amplia gama de funcionalidades, que incluyen la gestión de habitaciones por piso, informes para la limpieza de mucamas, gestión de bar y restaurante, cargos individuales por habitación, escritura de tarjetas magnéticas, entre otras.

## Funcionalidad

### Gestión de Reservas
- **Creación y Edición de Reservas:** Incluye detalles como clientes, vehículos, organizaciones, cargos adicionales, y cálculo automático de cargos por noche extra.
- **Visualización de Reservas:** Con filtros por estado, fecha y búsqueda.

### Escritura de Tarjetas Magnéticas
- **Generación de Tarjetas Magnéticas:** Para el acceso a las habitaciones.

### Reportes
- **Reportes Variados:** Como reservas de habitaciones, ventas del bar, informes para mucamas, ocupación actual, ocupación mensual, clientes, organizaciones, etc.

## Desafíos

A pesar del amplio alcance del proyecto, uno de los mayores desafíos fue la creación de una interfaz para comunicar el sistema web con el dispositivo de escritura de tarjetas magnéticas. Este dispositivo se conectaba a través de un antiguo puerto serial, y desde el sistema web era necesario enviar los códigos de las habitaciones encriptados en las tarjetas.

## Conclusión

**Sistema Hotelería** representa una solución completa y eficiente para la gestión hotelera. La variedad de funcionalidades ofrecidas y la capacidad de integrarse con dispositivos externos, como el dispositivo de escritura de tarjetas magnéticas, demuestran su versatilidad y adaptabilidad. Este proyecto destaca la habilidad para abordar desafíos técnicos complejos y proporcionar soluciones innovadoras para las necesidades de la industria hotelera.
